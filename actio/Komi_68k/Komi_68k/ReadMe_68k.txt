This is a quick'n'dirty compile of the SDL game 'Komi the spacefrog' v1.04 by
Alan Crossman.

Komi the spacefrog URL: http://komi.sourceforge.net/

All original source is in the src folder together with the original makefile. 

To compile the game for OS3, make sure you have your SDL environment setup
properly and use the Makefile.aos3 to compile (rename to 'Makefile').

Dernière version en téléchargement :

    • source - format tar.gz
    • Binaire RPM (Linux) - Le plus simple à installer si vous voulez juste jouer !
    • Source RPM (Linux) - Comme son nom l'indique...

Attention, le jeu n'est pas terminé, il manque des animations de personnages, des musiques, des bruitages, des instructios de jeu (mais elles sont sur le site) et des nouvelles fonctions, mais bon, c'est déjà tout à fait jouable...
Pour la version Windows, SDL et SDL_mixer sont inclus dans le fichier, donc vous n'avez besoin de rien de plus.
Pour compiler à partir du source, vous aurez besoin de la bibliothèque SDL, ainsi que de SDL_mixer. Une fois les deux installées, décompressez l'archive («tar -xzf BomberInstinct-xxx.tar.gz») puis dans le répertoire BomberInstinct tapez « ./configure » puis « make » et enfin « make install » (en tant que root) pour installer le jeu. L'éxécutable s'appelle «bi»; l'éditeur de niveaux «nived» et l'éditeur de sprites «spred» sont fournis avec le programme.

Yet Another SFCave 1.0


HOW TO PLAY

Click and the worm goes up, release and the worm goes down. Don't touch 
the borders or the obstacles in green.


ABOUT SFCAVE

The home page is hosted by sourceforge, you will find all the infos 
you need at 
http://yasfcave.sourceforge.net

SFCave was originaly available for Palm Pilot. You can find it at 
http://hp.vector.co.jp/authors/VA003665/index-e.html

You can also find a java online version at 
http://www.liquidcode.org/worm.html


HOW TO COMPILE

In order to assure a good portability, YaSFCave uses SDL and SDL_image 
libraries. You can find the last versions at http://www.libsdl.org.

You will need the sources of SDL and SDL_image if you want to compile 
YaSFCave. 

This software also use SFont. The files of this library are already included 
in this package, but you may want to compile with a newer version. You will 
find the SFont sources at http://www.libsdl.org/libraries.html.

All these links and more are provided here 
http://yasfcave.sourceforge.net

DPTEX.WAD v1.0 (18.4.98)


The DPTEX.WAD file puts the missing textures/patches back into DOOM.WAD
to make it work with Adoom.


To use it, just put the file into your \doom directory,
and then at the command prompt type:

Adoom -file DPTEX.WAD [the other wad(s) you want to add go here]

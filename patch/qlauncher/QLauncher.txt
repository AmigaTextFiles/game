o Description
  This is an another GUI utility for Quake.
  BUT! It uses standard GadTools library for GUI, so you can start
  it without startup-sequence, too!
  You can save your prefs!

o Installation:
  Move QLauncher and QLauncher.info files into Quake main directory.

o Usage:
  Start program from icon or from CLI.
  Set options and save it!

o Buttons:
  LOAD    - load prefs
  SAVE    - save prefs
  EXIT    - exit program
  LAUNCH! - start game

  Other buttons are standard quake commands.

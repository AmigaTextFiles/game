Read first
----------

Use this patch to upgrade your registered version 1.3 of Fields
of Battle to version 1.31.

Do NOT use if:
  - You only have a demo version
  - You have version 1.31 or newer


The patch consist of the following files:
-----------------------------------------

  fob13to131pch1.lha
  fob13to131pch2.lha
  fob13to131pch3.lha
  fob13to131pch4.lha
  fob13to131pch5.lha

Installing the patch:
---------------------

1.  Extract the patch archives (part 1,2,3,4,5) inside the Fields
    of Battle drawer.
      (This is the drawer where the file "keyfile" is.)

2.  Execute the batch file "update", type this command in your
    shell:

       execute update

    or

       double-click the icon "update"

3.  Delete the files:

    update and update.info

4.  Now you should be ready to run Fields of Battle v. 1.31



Changes:
--------

Fields of Battle, Version 1.31, changes.

a) The online help and encyclopedea has improved. Errors has been
   corrected and some historical documents has been added.

b) Bugs fixed:
    - UUencoder in the play-by-mail system has been corrected.
    - Minor bugs fixed.

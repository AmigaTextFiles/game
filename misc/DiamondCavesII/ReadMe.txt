   Diamond Caves II
------------------------
Version 1.4 (26.12.1997)

 Diamond Caves II can be run directly out of its directory. To install
 Diamond Caves II to (hard)disk, just copy the whole directoy (including
 all the subdirectoys) to the destination volume. If you are using the
 Workbench, drag the DC2 drawer where you want it.

 If you have already installed a previous version of Diamond Caves II,
 please use the update script for the Commodore Installer. This script
 will update all the necessary files without changing any of your Levels
 or Settings.

 For more information about Diamond Caves II or how to register, please
 read the english docfile 'DiamondCaves.guide'.

Pacman History, Version 1.
Author:           Jeroen Knoester
Creation date:    03/07/2000

Legend:

C - Gamma release. Should be stable.
B - Beta release. Playable, but might contain (minor) bugs.
i - Internal release. Not released to public. Can contain a number of bugs.

Changes from 0.8i to 0.9B:

- Added version diplay. Displays : Version number.
        ( i.e. 'v 0.9B' )
- Detects if the amiga is in PAL or NTSC mode.
- Added support for multiple difficulties. Somewhat buggy.
- Fixed a number of minor bugs.
- Fixed some minor bugs in ghost state switching.

Changes from 0.7i to 0.8i:

- Added title screen.
- Added Game over 'sequence'.
- Added various other presentation related items.
- Fixed several bugs in the ghosts routing code.
- Implemented many new ghost-states.

Changes from 0.6i to 0.7i:

- Cleanup of code, added several basic functions.
- Changed the behavior code for the ghosts into a limited state machine.
- Switched from sprites/bobs to bobs only.

Changes from 0.5i to 0.6i:

- Translation version. Translated code from AMOS into Blitz basic.
- Cleaned up ghost behavior code.

Changes from 0.1i - 0.45i to 0.5i:

- These are experimental AMOS versions. Ghosts behavior was seriously
   flawed.

End of File.

Gorman Fishtank **Spectrum Edition**
------------------------------------

This is a very early conversion of the Fishtank to the Spectrum emulator,
"Speculator '97".  To run the program:

 - Copy the contents of the "Scr" drawer to RAM:
 - Load Speculator
 - Type Load "" (the speechmarks are accessed by pressing right alt and p)
 - Type Run

The cursor is moved with the standard spectrum keys; q,a,o and p.


Requirements
------------

 - Speculator '97
 - An Amiga or compatible capable of running Speculator '97

 This allows you to eat Shmolluxes. With it, you are unvulnerable.

 It gives you a new life.

Shield
 Time (sec.)
 25
 With this shield, you can't be eaten by the Shmolluxs, but this does not mean _you_ can eat them.

Armaggedon (stockable)
 -
 pick: 50 ; use on hut: -50 ; use elsewhere: 0
 This bonus kills all Shmolluxs in the map. Moreover, if you use it on a Shmollux's hut square, it will destroy it. Very powerfull bonus.

Shock wave
(stockable)
 Range of action
 pick: 25 ; use: 50 by shmollux killed
 The paremeter is given in game units : one square equals to 3 game units. The shock wave kills all the Shmolluxs that are in its range of action.

Time stopper
 Time (sec.)
 pick: 25 ; use: -50
 This special item stops the game countdown for a few seconds, but does not stops the shmixgum countdown.

UltraPoints
 Time (sec.) 25
 If you eat this one, the Dituboites will give you 10 points instead of 1 point.

Ice
 Time (sec.)
 25
 This bonus will freeze all shmolluxs on the map or in the huts, allowing yourself to eat them.

High speed
(Stockable)
 Time (sec.)
 pick:0 ; use: 50
 Makes you run very fast.

Luck
 Time (sec.)
 50 + bonus chosen points
 This special item will give you a random bonus in the list of good bonus, except the shockwave, life and luck. The parameter (time) will be passed to the new bonus if it needs it.

Ceiling
 Time (sec.)
 75
 Fear this one ! It will propulse you to the ceiling, where you'll not be able to eat Dituboxes or to eat Shmixgums. But you will still be in danger because the Shmollux can eat you.
Deep fog

 50
 This malus will make you unable to view clearly the world. It creates a gray fog in the map.

MalusMap

Negative Dituboxes

 0
 Instead of receving points when you eat dituboxes, you will loose them (-1 point per ditubox).

Freeze

Very very dangerous.

Low speed

The Blue Mango Quest 0.6.3
September, 3rd. 2001 Release

Release Notes

The release notes for 0.6.2 were.... very very short. This time, I'll make them larger ;)
This release adds some major new features and major bugfixes. We are close to 0.7.0 features, so the next version should be 0.7.0.

Please remove your existing configuration file (~/.mangoquest/game_options in linux or mangoquest.ini in windows) for better default settings.

Windows window/fullscreen switching is disabled in options menu on Win32 platform.

Installation (linux users)
Download mangoquest-0.6.3.tar.bz2
Uncompress the archive, build, optionnaly uninstall previous version, and install.

tar xvyf mangoquest-0.6.3.tar.bz2
cd mangoquest-0.6.3
 ./configure
 make

and as root:
make uninstall (if you have installed previous version of mangoquest)
make install

Then launch mangoquest or mangopeeler as normal user.

If you can give us packages (RPM or DEB), email me .
 
Changes summary
MangoQuest: Added menus (including in game map selection), customizable number of lifes, winning post, new sounds, new textures, memory cleanups, updated game and editor documentation, added new maps, minor gameplay enhancements, and miscellaneous bugfixes (including timer fixes and better mouse smoothing).

See changelog ( http://mangoquest.sourceforge.net/ChangeLog-0.6.3 ) for more details.

Configuration files
You can edit game configuration in the "Options" menu of MangoQuest.
If you want to edit it by hand, you'll find it :
- linux users:  ~/.mangoquest/game_options file for mangoquest and ~/.mangoquest/editor_options for mangopeeler
- windows users: mangoquest.ini and mangopeeler.ini

New mouse smoothing system
Mouse smoothing could be ugly in 0.6.2 if your computer was a bit fast or if you had no luck. This is now fixed ! Go in the Options menu, enable " Smooth mouse" and modify the value with the mango. The more the value is, the more it's smoothed. But beware : if you increase mouse smoothing, you'll have to decrease "Mouse sensivity".

Keys
If you're lazy and you haven't read the game manual, here is a summary of the keys in MangoQuest 0.6.3 :
Step left: LEFT arrow
Step right: RIGHT arrow
Walk forward: UP arrow
Walk backward: DOWN arrow

Look around: mouse motion
Center the view: END key (close to PAGE UP and PAGE DOWN keys)

Options menu (in game): F10
Quit map: ESC

Use stocked bonus: see 6.1 section of MangoQuest user's manual.

The Blue Mango Quest 0.6.2
August, 14th. 2001 Release

Release Notes


  • General notes
Added mouse support, sounds, configuration files, on-HUD messages, new HUD graphics, updated game documentation, added new maps, and fixed miscellaneous bugs (including black-HUD bug on some video cards, a compilation problem with gcc 2.96 and packaging updates).

See changelog ( http://mangoquest.sourceforge.net/ChangeLog-0.6.2 ) for more details.

  • Packages
See download page : http://mangoquest.sourceforge.net/downloads.php .
We removed the data-only and src-only packages, since you need both the new datas and new sources.

Release Notes

Well, I said in 0.6.3 release notes that next version should be 0.7.0. But it's not.

Changes summary

See changelog ( http://mangoquest.sourceforge.net/ChangeLog-0.6.4 ) for more details.

How to use the Alternate Controls
- Launch the game once to write the correct config file for 0.6.4.
- Edit this config file :
  - linux users:  ~/.mangoquest/game_options
  - windows users: mangoquest.ini
and change alternate_controls=0 to alternate_controls=1 at the very bottom of the file.
- Save the file, and play.

What is the --nosound option
If you launch game with --nosound, your soundcard will not be used even if you have SDL_mixer installed. This can be useful for those who have problems with their drivers.

What is the Strategic Minimap dran on HUD ?
Press TAB key to see whet is it, and press TAB again to hide it.
Player is the blue triangle that rotates and move when you rotate and move.
Shmolluxes are the red losanges.
ShmixGums are the green triangles.

Keys
There is some new keys in version 0.6.4.

(NEW) In-game menu:ESC
(NEW) Show/Hide Strategic minimap on HUD: TAB

- Default contols
Step left: LEFT arrow
Step right: RIGHT arrow
Walk forward: UP arrow
Walk backward: DOWN arrow

Look around: mouse motion
Center the view: END key (close to PAGE UP and PAGE DOWN keys)

- (NEW) Alternate contols
Rotate left: LEFT arrow
Rotate right: RIGHT arrow
Walk forward: UP arrow
Walk backward: DOWN arrow
Look up/down: Page UP/DOWN and center with End

Use stocked bonus: see 6.1 section of MangoQuest user's manual.

Installation (linux users)
Download mangoquest-0.6.4.tar.bz2
Uncompress the archive, build, optionnaly uninstall previous version, and install.

tar xvyf mangoquest-0.6.4.tar.bz2
cd mangoquest-0.6.4
 ./configure
 make

and as root:
make uninstall (if you have installed previous version of mangoquest)
make install

Then launch mangoquest or mangopeeler as normal user.

Configuration files
You can edit game configuration in the "Options" menu of MangoQuest.
If you want to edit it by hand, you'll find it :
- linux users:  ~/.mangoquest/game_options file for mangoquest and ~/.mangoquest/editor_options for mangopeeler
- windows users: mangoquest.ini and mangopeeler.ini

man nehme
  • eine Region A mit Holzfällern [hoLz]
  • eine Region B mit Hasen oder so...
  • eine Region C mit Waffenbauer [waba]
  • und einen Transporteur [tran]


ein STEP definiert immer die Befehle für eine Einheit pro Woche.

STEP's für die Holzfäller 
#step gib [tran] alles holz
gib tran alles holz ; alles übergeben
mache holz ; und weiter roden
#step warten auf [tran]
mache holz ; wichtig das immer ein default Befehl gesetzt wird
; da er beim erzeugen gelöscht wird (kann ja sonst was für Müll
; von Eressea kommen)
; [tran] ist in Region B (Hasen jagen)
#step weiter warten
mache holz ; in dieser Runde übergibt [tran] das Holz and [waba]
; in der Region C
#step weiter warten
mache holz ; hier der letzte step
; [tran] ist wieder in Region B (Hasen haben aufgerüstet!)


STEP's für den Transporteur 
#step alles holz von [holz] holen
nach Region B
#step in Region B (Hasen jagen)
nach Region C
#step in Region C
gib waba alles holz
nach Region B ; und zurück
#step in Region B
nach Region A ; vor Hasen wegrennen


STEP's für [waba] sind überflüssig, da alle vier Runden der gleiche STEP wäre 
#step
mache waffen ; für hasen
#step
mache waffen ; für hasen
#step
mache waffen ; für hasen
#step
mache waffen ; für hasen
kann gleich in der Vorlage gesetzt werden



ein STEP beginnt immer mit STEP, Groß/Klein ist egal danach kann etwas beliebiges folgen, wird aber nicht weiter beachtet.

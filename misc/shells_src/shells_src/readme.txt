Shells Game Source Code Readme
==============================

 This package contains the complete source code of the Shells game for Amiga without the animation image sequence.
Animation image sequence can be found in the binary archive.

 Binary archive can be downloaded from Aminet:
http://aminet.net/package/game/misc/shells


Requirements for compiling
--------------------------
 You need to set up AmigaE compiler package to compile the source code. More information about AmigaE:
http://strlen.com/amiga-e


Compiling the sources
---------------------
 To compile the source code both source code files must be compiled and linked together. Using the original AmigaE
compiler this can be achieved by executing the following commands:

ec shellsgui
ec shells

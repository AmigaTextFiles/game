Benodigdheden

... om de geïnstalleerde programma's te draaien

  • CPU 68000 (68010+ is aanbevolen omdat sommige installs de VBR nodig heeft voor de 'stop via een toets' kenmerk)
  • Kickstart 2.0 (versie 37+)
  • Minimum van 1.0 MB RAM (soms meer, het ligt aan het geïnstalleerde programma)
Kickstart Emulatie

Sommige installaties gebruiken een zogeheten kickemu om een complete AmigaOS omgeving te hebben voor het geïnstalleerde programma. Voor die kickemu is een originele kickstart image nodig. Deze image kan gehaald worden uit een echte Amiga met programma's zoals GrabKick of iets vergelijkbaars welke gevonden kan worden op aminet. De ondersteunde imageversie ligt aan het geïnstalleerde programma. U kunt de informatie vinden welke image gebruikt moet worden in het ReadMe bestand. De image van kickstart versies 1.2 en 1.3 kan alleen gehaald worden vanuit een geschikte A500, voor versie 3.1 kan het gehaald worden vanuit een A500/A600/A2000, A1200 of A4000 welke uitgerust is met een Kick3.1 ROM. Als de image niet gevonden kan worden door WHDLoad of als de image de verkeerde versie heeft of veranderd is, zal WHDLoad stoppen met een geschikte foutmelding. Tevens moet er een relocatiebestand met de extensie .RTB gelijk aan het kickstart image aanwezig zijn. Dit relocatiebestand kan gevonden worden op aminet in het archief util/boot/skick346.lha. Beide bestanden, de kickstart image en de relocatiebestand moeten in de directory Devs:Kickstarts/ staan en de juiste naam hebben. Vanaf WHDLoad versie 16.5 kunnen ook de kickstart images van de Cloanto Amiga Forever CD's gebruikt worden. De images moeten hernoemd worden volgens het tabel onderaan. Tevens zal de rom.key gekopiëerd moeten worden naar Devs:Kickstarts/. Sommige oudere installaties hebben de kick40068.A1200 image nodig, deze installaties werken niet met de Cloantos ROMs.
Amiga Forever 2008 bevat ook de kick40068.A1200 en de images worden automatisch gekopiëerd naar de juiste plek the images.

... om nieuwe Slaves te schrijven / programma's te installeren

  • Aanbevolen is een 68030+ met MMU om alle kenmerken van WHDLoad te gebruiken (een krachtige machine is altijd goed om software te ontwikkelen ;-))
  • machinecode, assembler, amiga hardware kennen ...
  • een assembler (moet kunnen optimalizeren)
  • een paar andere programma's zijn handig (hexedit, disasm, ...)
  • wat meer MB's RAM
  • toegang tot verschillende machines om te kunnen testen (68000 - 68060, OCS, AGA, GfxCard)
  • een hardware of software freezer zoals AR3, HRTmon, TK
  • een NMI knop om een lopend programma te storen of om een software-monitor/freezer te betreden

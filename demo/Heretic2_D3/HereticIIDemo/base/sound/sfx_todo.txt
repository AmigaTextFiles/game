Heretic II Sound Effect Task List

This document is for the use of Heretic II programmers to list all sound effect slots which
are coded and awaiting entry of a sound effect wav file. Entries will be deleted from this
 document when the audio dept. has created a working sound effect for that file.

The following information should be included:
------------------------------------------------------------------------------
Name and location of file in code 
General description of sfx use 
Name of programmer
Date

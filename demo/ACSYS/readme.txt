System requirements:
--------------------
MC68020 or higher, OS 2.0, AGA, 2 MB Chip, 4 MB Fast


Installation:
-------------
Copy the complete directory on your harddrive and make
a assign to ACSYS1:. Now you can start the demo with a 
doubleclick on the ACSYS-icon.


Options menue
-------------
In the first line you choose the difficultly mode between
EASY, NORMAL and HARD. At the EASY-mode only you can see
1/3 of the demo level. If you play the HARD-mode you can see
all of the demolevel (2/3 of the original length).

The SOUND, ANIMATIONS and PASSWORD function is not available.

At the control menue you can put your joystick-, gamepad- and
keyboard configuration.  
The standard adjustment is:

Joystick/gamepad- move, fire, rotation on port 1
                  fire and rotation (with 1/25 delay) on button 1
				  weapon select with the space key

If you go to HIGHSCORE you can see the current HIGHSCORE list
for every difficultly mode. Please choose the mode with the
arrow buttons. The save function is not available.
With the right arrow button you can reset the list.

GAMEKEYS is a small summary to see the standard keys.

Right X-button = exit to OS 
START = start the game

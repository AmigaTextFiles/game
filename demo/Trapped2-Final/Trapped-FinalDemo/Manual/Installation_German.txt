Installation
------------
Legen Sie die Trapped II CD in Ihr CD-Laufwerk und waehlen Sie das
Trapped_Start Icon aus.

Sie werden nach dem Speicherpfad gefragt, wo der Spielstand gespeichert
werden soll. Dieser wird ca. 6MB auf Ihrer Festplatte belegen. Es ist 
notwendig, dass sie den vollen Pfad angeben, d.h. als erstes klicken Sie
auf Volumes, dann geben Sie den Speicherpfad ein.

Als naechstes erscheint das Start Menu:

Startup-Menu
------------

Start a new game        Load old Game

Gfx_Mode                Menu Mode
( ) OCS/ECS             ( ) Standard Pal
( ) AGA                 ( ) Double Pal
( ) Grafitti
( ) Picasso96

Precalculations
( ) Filtering           ( ) MipMapping

              Animation_Mode
                ( ) on/off


Waehlen Sie Start a new game, wenn Sie ein neues Spiel beginnen moechten.
Achtung, Ihr alter Spielstand wird geloescht.
Wenn Sie ein neues Spiel gestartet haben, werden Sie nach dem gewuenschten
Schwierigkeitsgrad gefragt. Hier stehen Ihnen 3 zur Verfuegung: 
Anfaenger, Abenteurer und Held. Als Anfaenger haben Sie es etwas leichter
und als Held natuerlich etwas schwerer.

Der GFX_Mode ist fuer die 3D-Engine. Folgende Modi sind verfuegbar:
OCS/ECS, AGA, Grafitti Card, Picasso96. Sollten Sie eine Cybervision3D
besitzen, benoetigen Sie die P96Emul.Library. Starten Sie das Spiel
im Picasso96 Modus.
Grafikkarten im allgemeinen werden mit den Treibern Picasso96 unterstutzt, 
welche im Aminet erhaeltlich sind.

Fuer den Lade-Bildschirm benoetigen Sie den Menu-Mode. Wenn Sie auf einem
Monitor spielen, der kein PAL darstellen kann, sollten Sie Double Pal fuer
den Menu-Mode waehlen.
Wenn die DoublePal ausgewaehlt haben, muss dieser Treiber in Ihrem
Monitors Directory installiert sein.
(Achtung: die 3D-Engine wird weiterhin in Pal laufen, ausser bei
 Grafikkarten-Modi)

Die Vorberechnungen (Precalculations): Sie koennen einige Funktionen der
3D-Engine von vornherein abschalten, da diese eine lange (bis zu 45 Sekunden)
Zeit benoetigen koennen, abhaengig von Ihrer Rechnergeschwindigkeit.
Hierbei handelt es sich um MipMapping und Texture-Filtering Vorberechnungen.

Der Animationsodus (Animation_Mode) entscheidet darueber, ob Sie wahrend 
des Spielens Animationen oder nur einige Bilder zu sehen bekommen.

PLAYING INSTRUCTIONS
--------------------


Left mouse button          -       Fires players bullets
Right mouse button         -       Reloads players gun
SHIFT key                  -       Fires missiles
P key                      -       Pauses game


EXTRA KEY FUNCTIONS

These following two options have been added to help people with 68000 Amigas

During play by pressing

S -  disables the superimposed score panel
B -  disables the background bullet detection

These functions should improve the game speed.

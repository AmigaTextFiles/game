History
~~~~~~~
1.3 (20.07.2014)
 - Fixed bug causing Atoms-X to quit immediately when return to start screen was selected during AI moves
 - Fixed bug causing full cells to be missed if quit requester was opened then cancelled during move
 - Cleaned up quit code so it works on every check, not just full cells
 - Added keyboard controls (space or enter to start, escape to quit)
 - Updated guide

1.2 (07.06.2014)
 - Fixed bug which caused native sound to be used even if disabled in the options
 - Removed FPU dependency
 - Added tooltype for manual sound control (SOUND=AHI|NATIVE|OFF|DISABLED)
 - Changed screen opening routines to be more friendly with hacks like VisualPrefs etc.
 - Added support for screenmode selection
 - Added tooltype for manual graphics control (NORTG and NATIVESCREENMODE=<ModeID>)
 - Added high resolution (640x480) support for AGA machines
 - Added 3.0/3.1 and NewIcons versions of icon
 - Added new option for keeping "excess" atoms after an explosion
 - Added new option for speeding up animation for slower machines, or disabling it altogether
 - Added option to save settings (player details, CPU level,  sound & graphics options) as tooltypes
 - Added checks so missing graphics files do not cause crashes

1.1 (26.02.2014)
 - Added support for classic chipsets by opening its own screen
 - Automatic graphics card detection
 - Added separate graphics for AGA and OCS/ECS machines
 - Added animation to exploding cells
 - GUI tweaks
 - Automatically selects the best graphics size for the machine
 - Removed requirement for zlib.library, jpeg.library and wizard.library
 - Added support for native audio (still uses AHI if available)
 - Added separate sound effects for classic sound support

1.0 (22.11.2012)
Initial release. Seems to be working fine here. Unable to get the colour remapping working properly, so it's graphics cards only for now.

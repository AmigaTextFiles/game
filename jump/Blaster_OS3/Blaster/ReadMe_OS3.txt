This is a quick compile for OS3 of the SDL game 'Blaster' v1.0.0 
by Oliver Pearce.

Concept and programming: Oliver Pearce
Some Graphics by Ari Feldman
Additional support: Stefan Strahm
Lakeside games website: www.lakesidegames.net
Blaster URL: http://lakesidegames.net/englischHP/BlasterMainEn.html

All original source is in the src folder together with the original makefile,
named Makefile.orig. 

Blaster is released under GNU GENERAL PUBLIC LICENSE, please read the
file 'COPYING' for more info. 

To compile the game for OS3, make sure you have your SDL environment setup 
properly and use the Makefile.aos3 to compile (rename to 'Makefile').

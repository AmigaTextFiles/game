This is a quick compile for OS3 of the SDL game 'JetPac' 
by James Blewitt. 

Jetpac - The Return is a remake of the original game Jetpac for 
the ZX Spectrum line of computers.

JetPac URL: http://web.ukonline.co.uk/james.blewitt/jetpac.html

All original source is in the src folder together with the original makefile,
named Makefile.orig. 

JetPac is released under GNU GENERAL PUBLIC LICENSE, please read the
file 'GPL.txt' for more info. 

To compile the game for OS3, make sure you have your SDL environment setup 
properly and use the Makefile to compile (type 'make' in the src folder).

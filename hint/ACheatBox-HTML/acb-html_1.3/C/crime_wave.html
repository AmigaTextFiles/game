Der Roboter ist nur an seinem Auge verwundbar, daher muß man im richtigen
Moment springen und eine Rakete abfeuern. Nach etwa vier Treffern sollte
das Problem gelöst sein. (Es geht auch mit normaler Munition, dauert dann
aber wesentlich länger.) Eine ähnliche Taktik sollte auch beim endgültigen
Endgegner angewendet werden.

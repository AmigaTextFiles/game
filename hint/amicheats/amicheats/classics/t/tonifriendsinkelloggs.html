Toni & Friends In Kelloggsland

CHEATS

On the main menu, type "ENGINEBYNEON". If it didn't work try typing "ENGINEBZNEON". Now use the following keys during play:

'F1'----------------For more energy
'F2'-----------------For more lives
'F3'------------------For more keys
'F4'-----------------For extra time
'F6'----------------For less energy
'F7'--To warp to the 1st boss level
'F8'--To warp to the 2nd boss level
'F9'--To warp to the 3rd boss level
'F10'------------For a higher score

On the main menu, type "SMACKSAREGREAT" to start with 6 lives instead of 3.

FUN

On the main menu, type "ILOVEKELLOGGS" to call up the sound fx and music test.

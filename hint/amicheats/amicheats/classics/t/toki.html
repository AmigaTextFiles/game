Toki

CHEATS

During play, type "KILLER" (the border should flash). This will give you infinite credits. Now use the following keys:

'R' ---- flips the screen upside down
'N' -------- to return screen upright
'F1-F7' - skip to corresponding level
'F8' ----------- end of game sequence

During the intro press 'ESC' then type "POORTOKI"

Start the game and press 'ESC' once the map screen has passed. Then pressing function keys during the intro will allow you to skip levels.

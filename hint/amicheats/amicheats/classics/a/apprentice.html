Apprentice

Produced by: Innerprise Software
Distributed by: Innerprise Software

DESCRIPTION

Jump'n Run Adventure! Lots of levels, including hidden levels and rooms, which can only be entered after solving different puzzles.

CHEATS

During play, press and hold 'DEL' and 'F3' for 50 guys and all the extras.

LEVEL CODES

On the title screen, press 'TAB' to go to the password screen. Now type any of the following level codes:

"WIZARD" | "GUILD" | "SPELLS"
"ARCANE" | "DRUID" | "FAERIE"

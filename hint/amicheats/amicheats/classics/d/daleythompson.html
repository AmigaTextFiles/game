Daley Thompson Olympic Challenge

CHEATS

On the high score table enter "HINGSEN.J" for your name and use the function keys to select events. If this does't work type "HINGSEN-J" then press the 'DEL' key. The high score table should say demo. Now type "-J" again and it should change to mega demo.

If you're having trouble getting this cheat to work then try this: On the title/high score page, wait until the music finishes playing. Now when you type in the cheat should hear a drum noise after each correct key press. If you hit a wrong key then you'll have to start again.

TIPS

For optimum results use the following shoes:

EVENT ---------- SHOES

DAY 1

Training -- training shoes
100m ------ running spikes
Long Jump ---- long spikes
Shot Putt ---- long spikes
High Jump ----- flat shoes
400m ------- running shoes

DAY 2

Hurdles --- short spikes
Discus ----- half spikes
Pole Vault - track shoes
1500m ------ track shoes

Use Lucozade only for the 400m and 1500m, assuming you only get two cans.

To get many points in the 100m event, run like mad until you're just about to cross the finishing line, instead wait for three minutes and then cross.

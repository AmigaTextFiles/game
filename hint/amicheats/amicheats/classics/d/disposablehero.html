Disposable Hero

CHEATS

First select the {OPTIONS} from the main menu. Then set the MODE option to {ARCADE} and set the sound sfx volume to zero. Next select exit. Now go to the highscore table, hold down the RIGHT MOUSE BUTTON and type "EUPHORIA". Then press the LEFT MOUSE BUTTON to go back to the main menu. Now select {OPTIONS} from the menu. When the options menu appears a new option will be present at the bottom of the list. Select this for a cheat menu.

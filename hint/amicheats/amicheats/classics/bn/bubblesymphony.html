CODES

Code to choose Normal Game or Super Game - "U$RJBL$D"

Code to start game as humans instead of dragons - "LDUJR$BR"

Old codes for Bubble Bobble do odd things. Try "BJBJBJ$P" (Original Game code) or "LBL$LJL$" (Shoes/Yellow Gum Code) and the game will begin with no background and remixed Bubble Bobble music!

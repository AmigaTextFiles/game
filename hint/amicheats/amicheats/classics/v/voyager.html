Voyager

CHEATS

Type "WHEN THE SWEET SHOWERS OF APRIL FALL" on the option screen. You will have access to the cheat menu.

Pressing 'ENTER' on the main option screen gives the following extra options:

'SHIFT', 'DEL', 'HELP'--Cycle through objects
'CURSOR KEYS'-------------------Rotate object
'7'-------------------Decrease size of object
'4'--------------------Enlarge size of object

IN GAME MODE

'F1'----------Go down a level
'F2'------------Go up a level
'F3'--Cycle through languages
'F8'------------Game Position
'F9'-----------Data on object
'F10'--------------Frame rate

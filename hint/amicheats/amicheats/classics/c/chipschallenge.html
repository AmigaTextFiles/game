"SAGITTARIANS MAKE BETTER LOVERS"------For infinite extras

NOTE: If you can't get the above to work try this;

Start the game normally then press 'F' (the screen should flip upside down):

"09/12/57" ----------------------- for unlimited time
"I THINK THEREFORE I AM." -------- and you don't need to collect chips to escape and you can press 'C' to skip to the next level

Pressing 'F' will return you to the game.

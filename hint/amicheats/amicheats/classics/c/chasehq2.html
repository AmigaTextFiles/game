Chase HQ II

CHEATS

Pause the game and type "IN A GARDEN IN". If this doesn't work try "INAGARDENIN".

'N'-------------------Skips levels
'1-6'--Jump to corresponding level
'W'-------------------Rocket thing
'T'----------------------More time

Hold 'H' and 'F5' while the game loads and all the roadside obstacles will be gone.

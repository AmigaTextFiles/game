Leatherneck

Produced by: MichTron/Microdeal
Distributed by: MichTron/Microdeal

DESCRIPTION

Storm the beach to rescue your buddies held captive behind enemy lines. Combat arcade game, 1-4 players.

CHEATS

Pause the game and type "CUTGBERTNECK" and press 'F3'. You are now invincible to the enemies, but not the other players. If this doesn't work try "CUTHBERTNECK".

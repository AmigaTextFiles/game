PASSWORDS

Enter "BIBBIB" for the password to play a hidden game.

CHEATS

During play, type "BRAINB" and press 'RETURN'. The message BRAIN BUG KEYCODES ENABLED should appear. Now use the following keys:

'F2'---------------Toggles invincibility
'1'-'8'--Jump to the corresponding level
'9'-----------------Jump to games ending
'1'-'9' on keypad---Warp to hidden rooms
'TAB'-----------------------Finish level

During play, type "MAMA" and press 'RETURN'. The message CHEAT KEYCODES ENABLED should appear. Now use any of the keys above and enter any of the following followed by 'RETURN':

"LIVES"----------------------For nine lives
"DIE"-------------------------Kill yourself
"SETCHK"------------------Set restart point
"SMARTS"----------------------For five film
"ITEMS"-----------------Gives you all items
"LEVEL"----------------------Complete level
"GAME"------------------Jump to game ending
"EMONST"-------Jump to end of level monster
"CREDIT"-----------------------Show credits
"VERS"--------------------Show game version
"DEBUG"-------------Shows debug information
"LEVRON"----------------Turns on all levers
"ADJUST"---Enable/disable frame rate adjust
"ADDWGT"--Enable/disable extra-enemy weight

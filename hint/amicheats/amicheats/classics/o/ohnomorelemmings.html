Oh No! More Lemmings

Produced by: Psygnosis Limited
Distributed by: Psygnosis Limited

DESCRIPTION

Just when you thought you were finally safe, those green-haired numbskulls have blissfully blundered off toward new perils. You'll have to save them again! Oh No! More Lemmings (Stand alone version) contains 100 brand new adventures for the mega-hit game Lemmings. Oh No! More Lemming (Add-on version) contains the same new adventures, but requires the original Lemmings program. Both versions feature mind-challenging game play and amusing graphics in the Lemming tradition.

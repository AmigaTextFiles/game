INSTALLING AND RUNNING PRINT MANAGER
~~~~~~~~~~ ~~~ ~~~~~~~ ~~~~~ ~~~~~~~
USING CITIZEN PRINT MANAGER FOR THE FIRST TIME
~~~~~ ~~~~~~~ ~~~~~ ~~~~~~~ ~~~ ~~~ ~~~~~ ~~~~
BEFORE LOADING CITIZEN PRINT MANAGER FOR THE FIRST TIME YOU SHOULD MAKE A
BACKUP COPY OF THE PRINT MANAGER DISK.  REFER TO YOUR AMIGA USERS MANUAL FOR
INSTRUCTIONS ON HOW TO PERFORM A DISK COPY.

WARNING: NEVER USE YOUR ORIGINAL DISK ALWAYS USE A COPY.


INSTALLING CITIZEN PRINT MANAGER
~~~~~~~~~~ ~~~~~~~ ~~~~~ ~~~~~~~
YOU MAY WANT TO RUN PRINT MANAGER FROM SOMETHING OTHER THAN YOUR BACKUP COPY
OF THE PROGRAM DISK; E.G. A HARD DISK OR YOUR WORKBENCH BOOT DISK PERHAPS.
 SUPPLIED IN THE "EXTRA" DRAWER OF THE CITIZEN PRINT MANAGER DISK IS AN
INSTALLATION PROGRAM TO ALLOW YOU TO DO THIS.  YOU MUST ENSURE THAT YOU HAVE
SUFFICIENT DISK SPACE (220K) ON THE DISK TO WHICH YOU ARE GOING TO INSTALL.
 IF YOU ARE GOING TO INSTALL PRINT MANAGER TO YOUR WORKBENCH BOOT DISK, REFER
TO THE AMIGA USERS MANUAL FOR INSTRUCTIONS ON HOW TO MAKE THE SPACE REQUIRED.


TO INSTALL PRINT MANAGER TO A HARD DISK
~~ ~~~~~~~ ~~~~~ ~~~~~~~ ~~ ~ ~~~~ ~~~~
1: BOOT YOUR HARD DISK AS NORMAL
2: INSERT THE CITIZEN PRINT MANAGER DISK IN ANY DRIVE
3: OPEN (DOUBLE-CLICK) THE "PRINT MANAGER" DISK ICON
4: OPEN (DOUBLE-CLICK) THE "EXTRA" DIRECTORY ICON
5: RUN (DOUBLE-CLICK) THE PMINSTALL PROGRAM ICON
6: SELECT THE REQUIRED LANGUAGE FOR THE INSTALL PROGRAM
7: AT THE PROMPT TYPE IN THE NAME OF YOUR HARD DISK AND PRESS ENTER.  FOR
EXAMPLE: DESTINATION NAME? SYS: [ENTER]
(SYS: IS THE DRIVE THAT THE SYSTEM HAS BOOTED FROM)
THIS WILL COPY PRINT MANAGER TO YOUR HARD DISK
8: WHEN THE INSTALLATION IS FINISHED PRESS ENTER AGAIN TO EXIT PMINSTALL
9: CLOSE THE "EXTRA" WINDOW
10: CLOSE THE "PRINT MANAGER" WINDOW
11: REMOVE THE DISK FROM THE DRIVE
12: THE CITIZEN PRINT MANAGER HAS NOW BEEN INSTALLED

CHANGING THE LANGUAGE
~~~~~~~~ ~~~ ~~~~~~~~
1: RUN THE PROGRAM AS DESCRIBED EARLIER
2: SELECT THE LANGUAGE YOU REQUIRE FROM THE FIRST SCREEN
3: SELECT THE SAVE ICON

PMprefs PROGRAM
~~~~~~~ ~~~~~~~
THE CITIZEN PRINT MANAGER REPLACES THE AMIGA PRINTER INTERFACE FOR TEXT AND
GRAPHIC FUNCTIONS.  PRINT MANAGER IS COMPLETELY COMPATIBLE WITH THE STANDARD
WORKBENCH PRINTER DRIVERS.  CERTAIN FUNCTIONS WITHIN THE PROGRAM HAVE BEEN
DISABLED, THESE APPEAR AS GREY AND HAVE NO ACTION.  FOR FURTHER INFORMATION
ABOUT THESE FUNCTIONS PRESS THE INFO BUTTON ON THE FIRST SCREEN OF THE
PMprefs PROGRAM.

THE MASTER MENU
~~~ ~~~~~~ ~~~~
PRINTER SETTINGS PAGE: CONTROLS THE PRINTER DRIVER BEING USED, THE PATTERN
USED IN PICTURES AND THE NUMBER OF PINS USED WHEN PRINTING AN IMAGE.

GRAPHICS PREFERENCES PAGE: DUPLICATES THE FUNCTIONALITY OF THE STANDARD
WORKBENCH GRAPHICS PREFERENCES PLUS A FEW ADDITIONAL FEATURES.  SEE YOUR
AMIGA USERS GUIDE FOR DETAILS ON THE USE OF PREFERENCES.

ADDITIONAL FUNCTION PAGE: AS THE NAME SUGGESTS, THIS MENU PAGE PROVIDES EXTRA
MISCELLANEOUS FUNCTIONS TO IMPROVE EASE OF USE.

COLOUR CONTROL PAGE: PROVIDES COLOUR AND BRIGHTNESS CONTROL FUNCTIONS TO
ENABLE THE USER TO ACCURATELY REPRODUCE SCREEN IMAGES ON THE PRINTER.

TEXT PREFERENCES PAGE: DUPLICATES THE FUNCTIONALITY OF THE STANDARD WORKBENCH
TEXT PREFERENCES.  SEE YOU AMIGA USERS GUIDE FOR DETAILS ON THE USE OF
PREFERENCES.

DEFAULT
~~~~~~~
THIS BUTTON RESTORES PRINT MANAGER TO ITS DEFAULT SETTINGS

LOAD : SAVE
~~~~~~~~~~~
THESE 2 BUTTONS ARE USED TO LOAD AND SAVE THE SETTINGS YOU HAVE CHOSEN
TO/FROM A STATUS FILE.  THIS FILE IS LOADED AUTOMATICALLY WHEN PMprefs IS
RUN.

OK + EDIT
~~~~~~~~~
THE OK BUTTON CAUSES THE CHANGES YOU HAVE MADE THROUGH THE MENU PAGES TO TAKE
EFFECT.  THE EXIT BUTTON CLOSES DOWN THE PMprefs PROGRAM, LEAVING PRINT
MANAGER INSTALLED IN THE MEMORY.  THE + BUTTON IS EFFECTIVELY THE SAME AS
PRESSING OK AND THEN EXIT.


FURTHER INFORMATION
~~~~~~~ ~~~~~~~~~~~

How to install Pinball mania

Make a directory and copy all the files from the 3 disks into the
directory.. ie: dh1:games/mania

copy the nonvolatile directory to sys:

copy all {disk 1}prefs/encarc to prefs/envarc

reboot

click icon in pinball mania

play the game.

What you will need:

CivHack does not place any huge overheads on your system:

  • gadtools.library (WB 2.0+)
  • asl.library (WB 2.0+)
  • Some civilization saved games (both CivAGA and ECS versions supported)
  • A copy of Civilization - note:CivAGA games are not compatible with CivECS and vice-versa
  • A harddisk is recommended but not essential

Memory usage is small (around 100k) and should not pose problems. Any 68k processor is fine, although a 68020 is recommended.

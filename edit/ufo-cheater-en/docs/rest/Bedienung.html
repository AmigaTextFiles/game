Usage
 

        Please read Protection advice before.
 

        At first you have to load a savegame. If you copied this tool to your
        UFO drawer, you have to change only the dir to one of the game_x
        drawers.

        Data of the game will now be read.

        Then the main window appears. There are 6 Buttons for the most important
        data of the game

                     Injury           : 0        soldier become healthy
                     Armour        : Flying suit
 

        BASES     :  At first appers a window with all basenames
                     Only existing Bases can be modified.
                     If you click the name you can choose to change
                     the installation or the storage.
 

          Installations : A window with the 36 fields of the base
                     will appear.
 
 AL = Access Lift  PD = Plasmadefence
 QU = Living Quarter   FD = Fusionsballdefence
 LA = Laboratory   GS = GravShield
WS = Workshop   MS = MindShield
sR = small Radar  PL = PSI Laboratory
lR = large Radar  HW = Hyperwave-decoder
MD = Missledefence  H1 = Hangarfield left above
ST = General Store  H2 = Hangarfield right above
AC = Alien Containment   H3 = Hangarfield left below
LD = Laserdefence  H4 = Hangarfield right below 

                     Click to a field and you can choose a building.

                     When you change a field the time to build is
                     set to 1. (technical reasons)

           Stores  : the buttons Weapons, Munition, other equipment and
                        craft equipment are explaning themselves

                     By clicking an object you can choose the value

        OTHERS :  the Money and the actual day can be changed

                     MONEY : the money is set to 1 000 000 000 through
                             a click (should be enough)

                     DAY  : a number from 1 to 31 can be choosen
 

        SAVE        :  saves the data back to the savegame

        ABOUT     :  info

        QUIT      :  you can leave this program (hey great)

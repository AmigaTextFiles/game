Usage

If you start the program first time, it starts in german. First choose a savegame-file an then go to prefs. here you can click on english and from now it will start everytime in english. The position of the savegame will also be saved in the prefs file, so everytime you start the program, it will try to load this file. You can change the savefile-position also in prefs.

The mainmenu

You see 6 buttons in a column. Please click on the button or scroll down the text for a description of the functions.

All data will be load when there is a need for. So if you click on bases, all data is loaded which is important for your bases, when clicking soldiers, all soldierdata is load. After closing the subprogram, all data will be saved back (after a requester)
 

soldiers

valuewindow

First there is the name of the soldier (the first ond only indicator for you). Then there are the values of this soldier. You can even change the armour and decrease injuries. With the 4 buttons named "S" you can save all values in one of the 4 seperate slots. With the 4 buttons named "L" you can load these saved collection of values from every soldier. With SAVE you can save the actual values to a special file for use with a couple of soldiers (read some rows down for more). With the X-button in front of the name you will go to the chooseewindow

choosewindow

To choose a soldier, click on the button before his name. With the arrows right you can scroll up and down (one or 5 soldiers once). Under the names are shown the numbers of the actual shown persons.
With the button actual 5 you can change the actual 5 soldiers (they are shown by names) to the values you saved in the valuewidow with SAVE.
With the button all you can do this with all 100 soldiers. (If there is now data saved with SAVE then the needed file doesn't exist and the values will be zero... all values will be zero...)
With the last button you can decrease injuries to 0 at every soldier.

financial

You can change money, the actual day and the doplomatical status of the countries. With this you can re-infiltrate a country (f.e.)

Spaceships

Since V2.10 up to 20 spaceships can be changed in their type and weapons. The only indicator ist the type of spaceship, which is shown top of the window. You can change it by clicking change (how easy). The change of weapons is the same. With the sliders you can put the value of amunition for the specified weapon.
With the X-button in front of the name, you can choose another ship.

Prefs
 

Here you can change the language to german or english. With the button Savefile the position (path) of the savegame can be specified new..

Bases

At first a window with 8 buttons appears. There you can choose one of your bases. Then you will see the base-window.

Buildings (installations)
Th base is shown schematic with shortcuts. When clicking a field you have to set a building (or even clear this field). In case of correct function in the game, all fields you change will get a buildingtime of 1 day. With the buttons load and save you can load or save the complete installations in 5 separate files. If you load an installationset, the buildtime is set to 1 day for all buildings.
 

buildtime
The buildtime of all building in construction is set to 1 day.

Weapons, Spaceship, Amunition, Equipment
All things in your base can be modified with the buttons. With the L and S buttons you can load and save the actual shown values in (from) special files.

file
Here you can save and load the complete base for later using. If you choose save, you can enter a coment. The actual values of the base will be saved.

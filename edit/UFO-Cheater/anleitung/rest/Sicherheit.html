Sicherheitshinweise

VORSICHT: Das Cheat Tool wirkt NUR mit einer der beiden ECS Versionen von UFO. Der gesamte Spielstand wird dabei in einer einzigen Datei namens savegame.sav abgelegt (evtl kommt noch der tacdata.sav hinzu, wenn ihr im Kampf speichert).

Die andere Version bzw die AGA Version (Die ich gerne hätte) speichen meines Wissens in vielen kleinen Dateien die base.dat, soldier.dat oder ähnlich heißen (verbessert mich, wenn ich falsch liege). Damit funktioniert dieses Tool auf keinen Fall. Für diese Versionen gibt es im Aminet aber genug Cheattools.

Das savegame.sav File muß eine Länge von 45299 Byte haben. Das Tool überprüft das gewählte File auf eine Länge von 45299.

Erstellt auf jeden Fall eine Sicherheitskopie des Spielstandes bevor ihr ihn ändert. Es ist bei UFO immerhin Platz für 10 Stück.

Im Kampf zu speichern und zu cheaten ist nicht möglich, da die Daten während des Kampfes nicht in die Datei savgame.sav, sondern in tacdata.sav gesichert werden.

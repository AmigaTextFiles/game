Bedienung

Beim ersten Programmstart muß eine Spielstanddatei gewählt werden. Das Programm startet nun in deutsch, um das zu ändern, muß auf Prefs und dann auf englisch geklickt werden. Die Spracheinstellung, sowie der Pfad des Savefiles wird gespeichert. Bei jedem Neustart des Programmes wird nun der angegebene Spielstand geladen.  Um die Spielstanddatei zu ändern ebenfalls in Prefs auf Savefile klicken und eine neue wählen.
 

Hauptmenü

Man klicke auf den Knopf, der einer näheren Beschreibung bedarf (oder scrolle im Text nach unten).

Das Tool unterteilt sich in mehrere kleine Programme, weshalb alle Daten erst eingeladen werden, wenn sie nötig sind. Nach dem Verlassen des Unterprogrammes werden die Daten dann zurückgeschrieben (optional natürlich).

Soldaten

Wertefenster

Ganz oben steht der Name unseres Soldaten. Nach dem Start des Programmes sollte das der erste Soldat sein. Danach kommen sämtliche Eigenschaftswerte, frei wählbar, versteht sich. Auch die angelegte Rüstung und die Verletzungen können modifiziert werden.
Mit den 4 L und S Knöpfen können die aktuellen Werte in eine von 4 Dateien geschrieben werden (mit S) oder die Werte aus einer der 4 Dateien können für diesen Soldaten übernommen werden. Der Knopf SAVE speichert in einer speziellen Datei, die benötigt wird, um mehrere Soldaten auf einmal zu verändern (gleich mehr dazu). Mit dem X-Knopf vor dem Namen wird das Soldatenauswahlfenster geöffnet.

Soldatenauswahlfenster

Um einen Soldaten anzuwählen genügt ein Klick auf den Knopf vor dem Namen. Unter den Namen werden die Nummern der sichtbaren 5 Soldaten angezeigt. Rechts neben den Namen befinden sich 4 Felder zum Scrollen (entweder um einen oder um fünf soldaten nach oben/unten)
Der Knopf aktuelle verändert alle 5 angezeigten Soldaten mit den Werten, die im Wertefenster mit SAVE gespeichert wurden.
Der Knopf alle verändert alle 100 Soldaten mit diesen Daten.
Mit heilen kann die Verletzung aller Soldaten auf 0 gesetzt werden.

financial

Hier kann das Geld, der Tag des aktuellen Monats und der diplomatische Status der Länder zur X-COM geändert werden. Man kann eine abtrünnige Regierung quasi re-infiltrieren.

Raumschiffe

Seit V2.10 können bis zu 20 Raumschiff in Typ und Bewaffnung verändert werden. Waffen und Schiffstyp können einfach per Klick auf einen der Wechseln-Knöpfe "rotiert" werden. Über die Schiebebalken kann ein Maximum an Munition eingestellt werden.
Der X-Knopf öffnet ein Auswahlfenster, um andere Raumschiffe zu ändern. Es gibt leider keinen Indikator, der die Raumschiffe identifiziert.

Prefs
 

Hier können die Sprache und die Position des Savefiles geändert werden.

Lager

Aus den 8 Knöpfen des erscheinenden Fensters wählt man sich ein Lager aus. Nun öffnet sich folgendes Fenster:

Ausbauten
Eine schematische Darstellung der gewählten Basis. Die Ausbauten können durch anklicken des betreffenden Feldes hinzugefügt oder auch entfernt werden. Eine Änderung eines Feldes führt zum setzten der Bauzeit von einem Tag (1 Tag ist leider nötig, um spielinterne Probleme zu vermeiden).

Bauzeit
Die Bauzeit aller in Konstruktion befindlichen Bauwerke wird auf 1 Tag gesetzt.

Waffen, Munition, Ausrüstung und Raumschiffsausrüstung
Alle in der Basis lagernden Dinge können geändert werden.

Datei
Hier können komplette Stützpunkte geladen oder gespeichert werden. Wird gespeichert, so kann ein Kommentar angegeben werden. Es werden die Daten gespeichert, die aktuell eingestellt sind.

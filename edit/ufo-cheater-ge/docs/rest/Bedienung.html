Bedienung

             Vor dem Weiterlesen unbedingt die Sicherheitshinweise beachten.

                Nach dem Start werdet ihr aufgefordert, einen Spielstand
                zu wählen. Wenn dieses Tool im UFO Verzeichnis ist, dann
                braucht ihr nur noch eines der Spielstandverzeichnisse
                zu öffnen und die Datei savegame.sav zu wählen.

                Die Daten des Spielstandes werden eingelesen und analysiert.

                Danach erscheint das Hauptfenster. Anhand der Knöpfe gelangt
                ihr zu den gewünschten Rubriken:

        SOLDATEN  :  Ein Auswahlfenster mit den ersten 20 eurer Krieger öffnet
                sich. Es können nur diese 20 Soldaten verändert werden, alle anderen
                im Spiel vorhandenen bleiben so wie sie sind. Ein Klick auf den Namen,
                und der Soldat wird optimiert; er bekommt folgende Werte:

                     Zeiteinheiten        : 200      sollte reichen
                     Gesundheit           : 255      Maximum
                     Energie                 : 189      mehr geht nicht
                     Reaktion               : 255      Maximum
                     Stärke                   : 127      mehr geht nicht
                     Schussgenauig.     : 255      Maximum
                     Wurfgenauig.        : 255      Maximum
                     PSI Stärke            : 127      reicht aus
                     PSI Fähigkeit        : 127      reicht aus

                     Verletzung            : 0          Heilung
                     Rüstung                :             Fluganzug

                     Der Energiewert ist unerklärlich, wurde aber getestet, der Stärkewert
                     wird ab 128 negativ (-127) , was wohl keiner will.

        LAGER :  Nach einem Klick erscheint ein Fenster mit den Namen eurer Lager.
                    Nur Lager, die existieren können verändert werden. Ein Klick auf einen
                    Namen öffnet ein weiteres Fenster. Hier entscheide man sich, ob der
                    Stützpunktaufbau oder der Lagerinhalt geändert werden soll.

         Ausbauten: Es öffnet sich ein Fenster mit den 36 Feldern des Stützpunktes.
                    Darauf ist die Belegung des jeweiligen Feldes zu sehen


AU = Aufzug FA = Fusionskugelabwehr
QU = Quartiere  PA = Plasmaabwehr 
LA = Labor  GS = Gravitationsschild
WE = Werkstatt GD = Gedankenschild
kR = kleines Radar  PL = PSI Labor
gR = großes Radar HW = Hyperwellendekoder
RA = Raketenabwehr  H1 = Hangarfeld links oben
LG = Lager H2 = Hangarfeld rechts oben
KL = Klimazelle H3 = Hangarfeld links unten
LS = Laserabwehr  H4 = Hangarfeld rechts unten

                     Ein Klick auf ein Feld öffnet eine Auswahl aller möglichen Bauwerke.

                    Wenn man ein Feld ändert, dann wird die Bauzeit für dieses Feld auf
                    1 Tag gesetzt. Das hat technische Gründe: ein hingeschummelter
                    HW-Dekoder funktioniert sonst nicht (ebenso Geankenschild).

          Lagerinhalt : Hier wird noch einmal zwischen Waffen, Munition, Ausrüstung
                    und Raumschiffausrüstung unterschieden.

                    Ein Klick auf den jeweiligen Knopf eröffnet ein Fenster, in dem die Dinge
                    im Lager und ihre Anzahl aufgeführt sind.

                     Ein Klick auf einen Gegenstand bietet die Möglichkeit, die Anzahl zu ändern.

                     Wer den Fluganzug vermisst -> Bugs + Fehler
 
                   Frage    : Warum ist die Auswahl der Menge beschränkt ?
                   Antwort : Ich habe die Erfahrung gemacht,(an mir selbst) daß, wenn man
                            dem User die absolute Auswahl überläßt, dann gibt jeniger welcher
                            häufig gerade einen Wert ein, der nicht den geforderten
                            Parametern entspricht. Bsp: es wird eine Shortint Zahl gefordert
                            (0-65535), dann kann man zwar die Eingabe auf 5 Stellen begrenzen,
                            aber der User gibt erst mal 99999 ein. Ich habe einige Werte
                            vorgegeben, die sich aus Erfahrung als nützlich erwiesen haben.
 

       SONSTIGES :  GELD und der aktuelle Tag können verändert werden.

                   GELD : das Vermögen wird auf 1 000 000 000 gesetzt
                                   (wird doch wohl reichen)

                   TAG  : eine Zahl von 1 bis 31 kann eingegeben werden

       SPEICHERN :  Speichert den geladenen Spielstand

       ABOUT     :  Was halt dort so steht

       ENDE      :  Na so was, man kann es auch wieder zumachen

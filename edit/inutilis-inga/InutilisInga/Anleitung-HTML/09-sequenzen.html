Sequenzen

1. Einleitung

Mit Sequenzen sind nicht die typischen selbstablaufenden Sequenzen in Spielgrafik gemeint, sondern Zwischeneinblendungen aus Bildern und ggf. Text. Dies kann für Intros, Extros oder Zwischenteile benutzt werden, oder aber auch einfach nur, um ein schönes Bild einzublenden.

Der Ablauf einer Sequenz wird in einer Textdatei (mit der Endung ".isq" im "Dats/"-Verzeichnis) gespeichert, die sehr einfache Steuersymbole enthalten muss. Zum Abspielen einer solchen Datei benutzt man den IngaSkript-Befehl "Sequenz" (siehe Befehlskompendium).

2. Aufbau einer Sequenzdatei

Eine Zeile einer solchen Textdatei besteht entweder aus einer beliebigen Textzeile, die unverändert gezeigt werden soll, oder aus einem Befehl, der mit einem Dollarsymbol ($) beginnt. Diesem Symbol folgt dann ein weiteres Zeichen, welches den gewünschten Befehl kennzeichnet, auf den wiederum ein Parameter folgen kann.

Es folgen die Befehle, wobei ein "x" für einen Parameter steht...

$:xZeigt ein neues Bild. x steht für den Namen der ibm-Datei (auch hier ohne Pfad und Endung!).
$!xSpielt eine Sample-Datei für z.B. Sprachausgabe ab.
$WxWartet x Sekunden.
$MWartet einen Mausklick ab.
$AxStellt die Art der Textdarstellung ein. Für x gibt es folgende Möglichkeiten:
0=Hochscrollender Text (wie im Kino!) (Voreinstellung)
1=Starrer Text am oberen Bildschirmrand
2=Starrer Text in der Bildschirmmitte
3=Starrer Text am unteren Bildschirmrand
$BxZeilenausrichtung 0=linksbündig (Voreinstellung), 1=zentriert, 2=rechtsbündig
$FxStellt die Farbnummer für den Text ein. (Voreinstellung: 255)
$OxGilt nur für scrollenden Text: Ändert die obere Begrenzungslinie. 0 bedeutet, dass der Text bis nach (fast) ganz oben gescrollt wird, 240 bedeutet, dass bis zur Mitte gescrollt wird, usw. (Voreinstellung: 0)
$UxGilt nur für scrollenden Text: Ändert die untere Begrenzungslinie. 480 bedeutet, dass der Text von (fast) ganz unten scrollt, 240 bedeutet, dass ab der Mitte gescrollt wird, usw. (Voreinstellung: 480)
$<xStellt die Einblendegeschwindigkeit ein ("Fade In"). Je höher der Wert, desto schneller wird gefadet. (Voreinstellung: 2)
$>xStellt die Ausblendegeschwindigkeit ein ("Fade Out"). Je höher der Wert, desto schneller wird gefadet. (Voreinstellung: 2)

3. Beispiel

Folgendes Beispiel würde zwei Bilder hintereinander zeigen und jeweils einen kleinen Text durchscrollen lassen. Der Text erscheint aber nur auf der unteren Bildschirmhälfte. Sobald der Text durchgescrollt ist, erscheint das nächste Bild. Das letzte $-Symbol hat eigentlich keine Auswirkung, aber dennoch ist es wichtig, da Texte immer erst angezeigt werden, wenn eine Nicht-Text-Zeile kommt.

$A0
$B1
$O240
$:Bild1
Es war einmal vor langer Zeit,
da ging die Welt unter.
Niemand hat's gemerkt,
und niemand hat sich dafür interessiert.
Schade eigentlich,
ein paar Dinge waren ja ganz schön!
$:Bild2
Auf einem ganz anderen Planeten:
Dort war alles schön,
die Aliens spielten,
die Mutanten zwitscherten,
und alle waren glücklich und zufrieden!
$

Der Entwicklermodus

1. Einleitung

Die Engine enthält einen Entwicklermodus, der einige "Cheats" ermöglicht. Er wird aktiviert, indem man das Merkmal "DEVMODE" im Piktogramm angibt. Außerdem muss das Spiel dann über die Shell gestartet werden, da ein Ein- und Ausgabe-Fenster benötigt wird.

Von der "SkriptShell" wird beim Kompilieren die Datei "SkriptShellLog" im "Dats"-Verzeichnis des Spiels erstellt. Dies ist eine Textdatei, welche eine Tabelle aller Sprungmarken mit ihren Adressen enthält. Diese sind wichtig für den Entwicklermodus!

Wenn man ein Spiel veröffentlicht, sollte man nicht vergessen, den Entwicklermodus auszuschalten!

2. Möglichkeiten des Entwicklermodus

2.1. Spielstände laden

Spielstände enthalten eine Adresse, die die aktuelle Position im Skript angibt. Wenn man das Skript jedoch weitergeschrieben hat, so ist es wahrscheinlich, dass die Adressen nicht mehr stimmen. Aus diesem Grund kann man im Entwicklermodus die Adresse neu eingeben (siehe Logdatei!). Die Zahl in der Klammer zeigt die alte gespeicherte Adresse an.

2.2. "Cheat"-Menü

Wenn man im Spiel die Taste <F10> drückt, so gelangt man in ein kleines Menü mit folgenden Möglichkeiten:

  • Sprung ausführen

    Man kann eine neue Adresse eingeben, zu der dann im Skript gesprungen wird (siehe Logdatei!)

  • Gegenstand aufnehmen

    Erzielt das selbe, wie der Skriptbefehl "InventarNehmen".

  • Variable prüfen/ändern

    Man kann den Wert einer beliebigen Variable ändern. In der Klammer wird der alte Wert angezeigt. Drückt man nur Return, so bleibt der Wert bestehen.

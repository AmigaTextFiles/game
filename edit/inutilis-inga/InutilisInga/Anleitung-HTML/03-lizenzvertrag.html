Lizenzvertrag

1. Haftungsausschluss

 Fehler in der Software können nicht ausgeschlossen werden. Dennoch übernimmt Inutilis Software / Timo Kloss keinerlei Haftung für Schäden, die die Software verursachen könnte. Dies gilt sowohl für das Inga-Entwicklerpaket an sich, als auch für Spiele, die mit diesem System entwickelt wurden.

2. Entwickler-Tools

 Der Benutzer hat ausschließlich das Recht zur Benutzung der Software ("InutilisGraphics", "SkriptShell", "PersonenEditor" und "OrtEditor") sowie die Datenträger erworben. Somit dürfen nur Sicherheitskopien für den eigenen Gebrauch gemacht werden, aber es ist verboten, die Software für andere Benutzer zu vervielfältigen! Es ist strengstens verboten, das Keyfile in irgendeiner Weise der Öffentlichkeit zugänglich zu machen oder es an einzelne Personen zu kopieren! Alle Rechte bleiben bei Inutilis Software / Timo Kloss.

3. Engine

 Der Benutzer hat das Recht, die Engine (auch die Versionen für andere Plattformen) mit seinen selbstentwickelten Spielen zu verbreiten. Solange der Gewinn an einem Spiel unter 150 Euro bleibt, muss kein Lizenzgeld an Inutilis Software / Timo Kloss gezahlt werden. Als Gewinn wird der Betrag angesehen, den das Entwicklerteam insgesamt erhält, ohne dass einzelne Löhne (z.B. für Grafiker, Musiker,...) abgezogen werden. Ist der Gewinn aber höher als 150 Euro, so müssen 20% des Gewinns abzüglich der 150 Euro an Inutilis Software / Timo Kloss gezahlt werden! Jedes Spiel, welches mit "Inga" erstellt worden ist, muss bei Inutilis kostenlos registriert werden. Außerdem muss das "Inga"-Logo ("Inga-Logo.iff"-Datei) z.B. im Intro jedes Spiels erscheinen, sowie auf der Verpackung (falls vorhanden) deutlich abgedruckt werden.

4. Updates

 Jeder rechtmäßige Besitzer des Inga-Entwicklerpakets erhält kostenlose Updates, solange sie zur Version 1.x gehören. Updates können von der Inutilis-Internetseite heruntergeladen werden, oder man kann sie auf Anfrage auf Diskette zugeschickt bekommen (ebenfalls kostenlos).

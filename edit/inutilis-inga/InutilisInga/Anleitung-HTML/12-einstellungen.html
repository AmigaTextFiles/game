Einstellungen für die Engine

Es wurde zwar versucht, die Bedienung der Engine so einfach wie möglich zu halten, jedoch kommt man um einige Einstellungen nicht herum. Es gibt dafür kein extra Konfigurationsprogramm, sondern lediglich einige Merkmale (ToolTypes), die im Piktogramm angegeben werden können:

"SOUND"Aktiviert die IngaSound-Erweiterung für Sound- und Sprachausgabe, falls sie überhaupt vorhanden ist.
"PLAYER"Die "medplayer.library" wird geöffnet, damit Modules abgespielt werden können.
"AUDIOCD"Die "cdplay.library" wird geöffnet, damit Musik-CDs abgespielt werden können.
"CDDRIVE=??"Falls CD-Musik abgespielt werden soll, so muss hier das CD-ROM-Laufwerk angegeben werden (z.B. "CD0:", "CD1:" usw.)
"GAMEPATH=??"Ändert den Pfad, in dem die Spieldaten gesucht werden sollen. So lässt sich die Startdatei (die Engine) auf Festplatte installieren (wodurch die Pikt.-Merkmale veränderbar werden), während die Spieldaten z.B. auf einer CD liegen. Die Spielstände werden jedoch im Verzeichnis der Engine gespeichert (also auf Festplatte).
"DEVMODE"Hiermit wird der Entwicklermodus aktiviert (siehe Kapitel "Der Entwicklermodus").
"MINFREE=??"Zur Angabe des mindestens frei zu haltenen Hauptspeichers in MB (voreingestellt ist 2).
"FPSCOUNTER"Aktiviert die Bilder-pro-Sekunde-Anzeige (nur IngaX und IngaOS4).

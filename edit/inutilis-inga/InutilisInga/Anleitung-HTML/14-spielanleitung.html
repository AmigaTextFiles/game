Spielanleitung

1. Grundsteuerung

Der Charakter wird ganz einfach mit der Maus gesteuert.

Linke MaustasteGehen, Benutzen, Nehmen, Reden mit... usw.
Rechte MaustasteAnschauen (und Sätze abbrechen)

Tasten:

<F1> und <F2>Zum Ändern der Texteinblendungsdauer.
<F3>*Zum Ein-/Ausschalten der Texteinblendungen, falls Sprachausgabe aktiv ist.
<F4>*Zum Ein-/Ausschalten der Sprachausgabe.
<F5>Menü
<F10>Nur im Entwicklermodus: Entwicklermenü
<ESC>Bricht selbstablaufende Spielteile ab, falls von der Situation vorgesehen.
*=Nur vorhanden, wenn "IngaSound" aktiv ist.

Beenden kann man das Spiel auf dem Menübildschirm (siehe Inventarleiste!).

2. Die Inventarleiste

Die Inventarleiste wird sichtbar, wenn man mit der Maus an den unteren Bildschirmrand gelangt. Alle Gegenstände, die man besitzt werden dort angezeigt. Ein rechter Mausklick auf einen Gegenstand lässt wie üblich Ermentrud einen Kommentar dazu offenbaren. Klickt man einen Gegenstand mit der linken Maustaste an, so bleibt dieser am Mauspfeil hängen. Klickt man dann auf etwas anderes, so wird beides miteinander benutzt. Dies gilt sowohl für Objekte am aktuellen Ort, als auch für Inventargegenstände. Klickt man zweimal auf einen Inventargegenstand, so benutzt man ihn allein. Um ins Menü zu gelangen (zum Spiel beenden, Spielstand speichern, usw.), muss man in der Inventarleiste sinnigerweise auf "Menü" klicken.

3. Das Dialogsystem

Es wird das "klassische" System von Grafik-Adventures benutzt: Es stehen mehrere Sätze zur Auswahl, die man nach eigenem Ermessen anklicken kann, um sie zu sagen.

4. Sequenzen:

Sequenzen (Bilder ggf. mit Texteinblendungen) können mit <ESC> komplett abgebrochen werden.

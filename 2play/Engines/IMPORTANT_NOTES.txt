~~~~~~~~~~
This game has only been tested on plain Amiga 1200, with and w/o 4 Mb of
fast-ram. AGA is required and the game has been developed without NTSC in
mind (works correctly only in PAL).


Disk-install
~~~~~~~~~~~~
Copy all the files to a disk labeled Engines. Make sure it has a boot block,
otherwise you can install one by typing 'Install DF0:' in a shell, if the
disk is in DF0:. Run the game by booting from the disk.


HD-install
~~~~~~~~~~
Copy all files and directories except the s drawer to a directory on your
HD. Start the game by double clicking the Engines icon from WB.

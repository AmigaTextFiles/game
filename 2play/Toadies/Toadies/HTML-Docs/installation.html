Installation INSTALLATION:

 Unpack the archive and copy the drawer "Toadies" wherever you want, e.g. "Games:". Toadies needs no assigns and uses no files outside this directory.

If you have a previous version of toadies installed and you want to keep your teams, skins and levels then install the new version to a different path on your harddisk and copy the contents of these drawers to it.

Requirements REQUIREMENTS:

 Hardware:
Any Amiga with at least 2MB chip and 4MB fast RAM. But don`t expect this game running smooth on an 68000. All I can say it runns well with all features enabled on my 68060/50MHz, but a 68030 should do it with some effects disabled.

If you are running this game under WinUAE anything below Pentium II / 300Mhz makes no sense.

! You should have ca. 1.5 MB of free chipram, otherwise it could crash !
Also you should not run it unless you have saved your work you might have done before one your machine. Expect that TOADIES trashes your memory.

Software:
No special software is required, except some system libraries like the math libs (I think).

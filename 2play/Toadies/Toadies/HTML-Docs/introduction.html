INTRODUCTION INTRODUCTION:

 "Toadies" are nasty little worms with a lot of nasty weapons.

This is another Worms clone game for the AMIGA. I think this is the only Worms clone for the Amiga that comes close to the original from Team17. In some aspects it even beats Worms (more nasty weapons, smooth scrolling etc.), in some not.
I did this in 1997, now I decided to upload it. Don`t know why I didn`t this earlier when the worms-fever on the Amiga came up, sorry folks !

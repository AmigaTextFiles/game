ASB version 0.4 from 28. April 2005
inspired by the cool game Air-Sea-Battle from 1981 for the Atari 2600.

The left tank is controlled with A,W,D, the right with the cursor keys.
The first player who shoots 25 planes scores.

The game is licensed under the GPL, EXCEPT THE FONT! The file astronbo.ttf is property of Ray Larabie, see http://www.larabiefonts.com for details.
If you want to make the game fully GPL compatible, replace the file astronbo.ttf with another true-type font and change the name in the source: asb.cpp, line 84 to 88.



Install instructions for Dwarf Connect 4. 

Windows Install
=-=-=-=-=-=-=

- Install via the msi found at www.dwarfcity.co.uk/games/connect4.php

- To build Dwarf Connect 4 on a windows machine you must have the following installed:

   = SDL (1.2.*)
   = SDL_Image 
   = SDL_Mixer
   = libpng
   = zlib
   
   Run the command to start the build process
  
   make -f makefile-win

   NOTE: The archive is set to build using the minGW compiler on windows systems.

#
# [ OVERVIEW ]
# This is a port to MorphOS of Stockfish 5.
# Stockfish 5 is an UCI chess engine and it is developed by
# Tord Romstad, Marco Costalba and Joona Kiiski
# http://stockfishchess.org/
# http://en.wikipedia.org/wiki/Stockfish_(chess)
#
# [ REQUIREMENTS ]
# - MorphOS 3.18 running on PowerPC based computers
#
# [ USAGE ]
# No particular installation is needed, is possible to run the application
# anywhere in your system.
#
# The engine can be used through the Shell interface, or with a GUI who supports UCI engines.
# From Shell:
# 1) > Stockfish5_MorphOS
# 2) enter your commands
#
# [ VERSION HISTORY ]
# - Version 5.0 released 19-Aug-2024, first public release
#
# [ COMPILING ]
# Stockfish can be compiled with gcc 9.5.0
# who must be installed on the system
# From Shell:
# 1) Enter 'src' directory with 'cd' command
# 2) Run 'make' command
#
# [ AUTHORS ]
# MorphOS port by
# Domenico Lattanzi   domenico.lattanzi@mailfence.com


XTR Editor 
==========


Create your own tracks, or modify existing tracks - all aspects can be
changed, including the graphics.

Fully OS3 compliant - yes, you can multitask and use it on gfx boards 
and set the number of colours to use etc.

Friendly and Intuitive to use - standard Amiga menus with keyboard 
shortcuts. Full set of preferences.

Include your own tile sets and objects - example palettes to be included 
in the distribution.

Track layout is done by placing tiles upon a simple grid.

Full on-line help included plus quick start guide.

Layout all the track boundaries, computer car routes, drop down points, 
surfaces (inc. jumps, water, etc), secret paths, etc.

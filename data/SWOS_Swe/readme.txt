This datafile contains the latest transfer and manager changes
in swedish domestic football.

I also support the teams that doesn`t play in the first division like:

Västerås SK
Ljungsskile


I use SWOS Ed 1.8 and as a result of that all player ratings are
value/skill compatible.

The datafile is packed using Propack/RNC.

The data file has been tested with the JST HDLoader and SWOS 96/97,
and seems to work properly in career mode.

Please send bug reports, suggestions, comments etc to "david.wahlberg@iname.com".

Redistribution

This page is intended to give GNU/Linux distribution builders some help in building packages of Armagetron for their system. For the average user, it is of little use.

This document is quite new so it may be of limited use. Please contact me directly ( ) or use the contact page to get help or get your suggestions or own build scripts integrated into the next version. After all, it is in my own interest to make your work as easy as possible so this little game gets on as many distributions as possible.

RPM package building

Have a look into the folder build/redhat of the source distribution. There is a script called build that can do all the work for you. You may have to adjust some of the variables set at the start to tailor some things. The user who executes the script must have write permissions to the RPM package directory tree, but does not need to be root.

Alternative: just use the distributed source RPMs.

INTRODUÇÃO
----------

FuhQuake, a partir do build 275, dá suporte a binds em todos o botões de um
mouse Logitech. Isso funciona com ou sem  -dinput na linha de comando. Todos
os 8 botões de um logitech mx500 foram testados com sucesso.


PREREQUISITOS
-------------
*) Você deve usar os binarios para win32 (build >= 275) e um mouse logitech
com o logitech mouseware instalado. Mouseware v9.73 funciona bem, já o 
Mouseware v9.75 *não* funciona. O recomendado é o Mouseware v9.73 (mas outras
versões podem funcionar).

Você pode baixar o Mouseware v9.73 em: http://www.fuhquake.net/files/misc/mw973enu.exe

*) Você deve ter o mw_hook.dll no seu diretório do quake (o mesmo em que estiverem
os executáveis do FuhQuake).


USANDO
------
Primeiramente use -m_mwhook na sua linha de comando. Ex.: fuhquake-gl.exe -m_mwhook -etc

Você agora pode usar MOUSE4 - MOUSE8 no seu quake.
(você vai ter que testar os botões para saber qual é o seu MOUSE4 etc).


CRÉDITOS
--------
-m_mwhook funciona interceptando as mensagens do mouseware enviados para o quake
e fazendo-os funcionar direito para que as mensagens sejam interpretadas pelo quake.

A idéia de fazer isso foi de [EZH]FAN.
Foi primeiramante implementado no qw262 de BorisU.

Todos os créditos devem ir para essas duas pessoas.

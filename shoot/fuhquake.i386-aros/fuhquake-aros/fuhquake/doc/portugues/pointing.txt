INTRODUÇÃO
----------
Você pode apontar para objetos e identificá-los e sua localização.
Diferente do Qizmo o FuhQuake *não* vai esconder suas mensagens de você.
Entretanto, o FuhQuake não permite o você reportar objetos que você não tem
um claro campo de visão dele (em outras palavras, você só pode reportar coisas
que você vê).


USANDO
------

$point / %x = item para o qual você está apontando.
$pointloc / %y = localização do item para o qual você está apontando.
$pointatloc / %t = mesmo que "%x at %y" 

Use o comando "tp_point" para especificar que objetos você deseja reportar.
O modo de uso é o seguinte:
tp_point <lista dos objetos que você deseja reportar>

A lista de objetos que você pode apontar é:

"quad", "pent", "ring", "suit", "ra", "ya", "ga", "mh", "health", "lg",
"rl", "gl", "sng", "ng", "ssg", "pack", "cells", "rockets", "nails", "shells",
"flag", "teammate", "enemy", "eyes", "sentry", "disp"

"powerups" é o atalho para "quad pent ring"
"ammo" é o atalho para "cells shells rockets nails"
"players" é o atalho para "enemy teammate eyes"
"weapons" é o atalho para "sg ssg ng sng gl rl lg"
"default" é o atalho para as configurações originais.
"all" é o atalho para todos os objetos.

Você também pode usar "tp_point +item" para adicionar algum item à lista.
Ex.: "tp_point +pent" adiciona o pent a sua lista.
     "tp_point -pent" remove o pent da sua lista.


Você pode usar 'tp_name_*' para alterar os nomes dos objetos que você quer reportar.


MODO DEATHMATCH
---------------
No dm, você não pode reportar armas se estiver jogando em dmm2, 3 ou 4
e você também não pode reportar ammo em dmm4. É dessa forma porque
reportar esse tipo de coisa é inútil nesses modos do jogo.


PLAYER POINTING
---------------
A partir do FuhQuake v0.26 você pode reportar pessoas do seu time, inimigos e os olhos.

Se você apontar para um inimigo isso vai identificá-lo com "enemy" (ou qualquer
coisa que estiver em "tp_name_enemy")

Se você apontar para um amigo isso vai identificá-lo como "teammate" (ou qualquer
coisa que estiver em "tp_name_teammate"). Uma coisa, se tp_name_teammate estiver
com o valor "" (que é o default) isso vai dizer o nome do jogador em vez de 
"teammate".

Se você apontar para os olhos (pessoa que está com o ring) isso vai identificá-lo
como "eyes" (ou qualquer coisa que estiver em "tp_name_eyes"). Você não pode
distinguir se os olhos são um amigo ou um inimigo.

Se o amigo/inimigo/olhos tiver com quad ou pent, isso ai reportar "quaded enemy"
ou "quaded pented eyes" etc. Você pode usar "tp_name_quaded" e "tp_name_pented"
para costomizar isso.


TEAMFORTRESS POINTING
---------------------
No teamfortress você também pode reportar dipensers e sentry guns.
O sistema de pointing não vai distinguir entre sentry guns e dispensers
amigos e inimigos. Use "tp_name_dispenser" e "tp_name_sentry" para costomizar
esses nomes.

Player pointing também funciona diferente no teamfortress. Você pode reportar
qualquer jogador mas você nunca pode dizer seus nomes (mesmo que ele seja do
seu time). A classe dele entretanto é dito. Por exemplo "enemy soldier" ou
"teammate demoman". Você não pode usar isso para detectar spys. Se você apontar
para um spy inimigo usando um disfarçe de demoman do seu time, o FuhQuake
vai reportar "teammate demoman" e não "enemy spy".

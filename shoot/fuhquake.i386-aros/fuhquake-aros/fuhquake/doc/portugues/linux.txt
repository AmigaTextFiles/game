INTRODUÇÃO
----------
Esse doc explica algumas coisas sobre os binarios para linux.


C LIBRARY VERSION
-----------------
Você deve estar usando libc6 (também conhecido como glibc2) para poder usar s binarios
do FuhQuake. Versões mais antigas não são suficiente. Para ver que versão você tem
digite 'ls lib/libc.so.6' e se o arquivo existir você tem o libc6. Todas as distribuições
mais recentes (cerca de 2anos) tem instalado o libc6 como padrão. Distribuições mais antigas
vem com o lobc5.


SVGALIB BINARY
--------------

Prerequisitos:
Para o fuhquake.svga funcionar bem depende escencialmente da configuração
do svgalib. A maioria das distribuições do linux colocam o config do svgalib
em /etc/vgs/libvga.config .


variáveis do svgalib:
vid_waitforrefresh: flips vsync on/off.
vid_redrawfull: redesenha toda a tela a cada frame (?).

linha de comando do svgalib:
-mode x : determina o mode de vídeo para x(use vid_modelist no console para ver o que x pode ser).
-w x , -h y, -d z : tenta usar o modo de vídeo width = x, height = y, depth = z .
-nokbd = desativa o suporte ao teclado.
-mdev dev = localização do mouse device.  eg -mdev /dev/mouse
-mrate rate = define mouse rate (padrão 1200).


X11 SOFTWARE BINARY
-------------------

Prerequisitos:
Ter X configurado e funcionando usando 8, 16 ou 24 depth.

linha de comando do x11 :
-verbose : diz mais informações.
-winsize x y : define o tamanho da janela em width = x, height = y .
-width x : especifica o width da janela.
-height y : especifica o height da janela.
-nomouse : desabilita o mouse.

Se o seu mouse está se comportamento estranhamente, tente toggling _windowed_mouse .


GLX BINARY
----------

Prerequisitos:
i)   Ter o GLX instalado (pegue-o na página da nvidia ou use o metasoft software
implementation). Obviamente precisa de uma placa nvidia para FPS decente.
ii)  Ter o X funcionando direito. Se você quer rodar o FuhQuake em fullscreen
então inicie o X na mmesma resolução que você quer rodar o FuhQuake.
iii) Possivelmente você vai precisar do X versão >= 4.0 e talvez você precise da 4.2.

linha de comando do glx:

-width x        : especifica o width da janela.
-height y       : especifica o height da janela.
-fullscreen     : roda em mode fullscreen (isso desabilita o alt-tab etc)
-nodga          : desabilita o DGA.  Use isso se você não quer o DGA input suport
                  ou sua versão do X não suporta DGA. Mesmo que -nodga -nokdga .
-nomdga         : desabilita DGA mouse input. Use isso se você está tendo problemas com o mouse.
-nokdga         : disable DGA keyboard input. Use isso se você está tendo problemas com o teclado.
-gamma x        : especifica o gamma.
-nohwgamma      : Desabilita o suporte ao gamma do hardware. Use isso se você tem problemas com o gamma.


Se o seu mouse está se comportamento estranhamente, tente toggling _windowed_mouse .

INTRODUÇÃO
----------

A parametro -ruleset da linha de comando é usado para forçar o ajuste a regras
de uma competição.


POSSÍVEIS RULESETS
------------------
Por enquanto só há o ruleset "default" e o ruleset "smackdown".


USANDO
------
Use "-ruleset <nome>" na linha de comando para usar um ruleset.
Se -ruleset não é especificado, então é usado o default ruleset.


SMACKDOWN RULESET
-----------------
O smackdown ruleset difere do default da seguinte maneira.

*) As variáveis tp_triggers, tp_msg_triggers, cl_trueLightning, scr_clock, r_aliasstats
são desativadas.

*) Se o servidor não tiver a chave serverinfo maxfps, então 77 fps é permitido ao invés
de 72 fps que é o limite default.


F_RULESET
---------
O FuhQuake (build >= 275) vai responder a f_ruleset quando um ruleset é usado.
No futuro isso pode ser removido e os rulesets serão simplesmente respondidos
em com o f_modified.

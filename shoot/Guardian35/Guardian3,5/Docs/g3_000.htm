Beschreibung


Bei diesem Spiel handelt es sich um ein horizontal scrollendes Ballerspiel. Hiervon gibt es allerdings schon mehr als genug, die technisch auch um einiges besser sind. Was aber bringt das, wenn man während einer 10minütigen Bildberechnung sich etwas die Zeit vertreiben will, aber natürlich das Multitasking nicht abgeschaltet werden darf.
In so einem Fall kann man jetzt einfach zum Guardian35 greifen, der auf langsamen Amigas zwar einiges an Prozessorleistung raubt, aber eben auch noch was übrig läßt. Zum Resourcen-Sparen gehört es dann auch, daß kein eigener Screen geöffnet wird, sondern daß das Gameplay ausschließlich in einem Workbench-Fenster stattfindet.

Gestartet wird das Spiel entweder über die Workbench oder per Shell. Die Steuerung erfolgt mit dem Joystick, abgebrochen wird jederzeit durch die ESC-Taste.

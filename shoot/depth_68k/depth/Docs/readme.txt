■Depth ver 0.02
2001/5/25

【概要】
どこかで見たことがあるような潜水艦ゲームです。

【動作環境】
SDLが動くことが必須。Linuxなど。

【同梱ファイル】
readme.txt	このファイル
Makefile	メイクファイル
*.c/*.h		各種ソースファイル
depthbmp/	グラフィックファイル

【動作方法】
■コンパイル方法
SDL1.1.8以降が必要です
makeと打つだけです。

■プレイ方法

$ ./depth
でプログラムが開始し、メニュー画面になります。
矢印キーで選び、'Z'で決定します
メニューは以下の通りです。
●Play
	ゲームを開始します。矢印キーで左右に敵弾をよけながら、潜水艦
	を破壊します。
	'Z'あるいは'X'で爆弾を投下します。同時投下数に制限があり、
	画面上の中央に数が示されています。
	画面右上のゲージはデコイ残弾数を表しており、最大10発まで備蓄できます。
	デコイは敵の追尾ミサイルの照準を狂わせることが可能で、'C'で発射します。
	デコイは一画面に一発しか撃てません。
	画面左上の三角は機体の速度を示しています。最大3段階まで高速化されます。
	真ん中下の黒い半透明ゾーンはレーダーです。赤い点が敵潜水艦を表します。

▼アイテムについて
	敵を倒すとアイテムが出ます。アイテムの色でそれぞれ効果が違います。
	青…機体速度を上げます。
	赤…爆弾の性能が上がります。
	緑…デコイを3発補給します。
	黒…爆弾の同時投下数を上げます。最大8個までです。

●Score
	ハイスコアラーを見ます。
●Exit
	ゲームを終了します。

ゲーム中にESCを押すと、いつでもプログラムを終了します。
F12を押すとスクリーンショットを撮ります。

▼注意事項
現時点で、ハイスコアは保存されません。

Utilisation:
    Avant chaque partie vous pouvez entrer votre nom en utilisant le joystick.
Haut/bas pour faire défiler les lettres, droite/gauche pour entrer la lettre
suivante. Appuyez sur "Fire" pour débuter la partie.

Deux mode de jeux sont disponibles.
En mode normal, les tableaux ce succèdent par ordre croissant de difficulté.
En mode crazy, les tableaux apparaissent aléatoirements ainsi que la vitesse.
Vous pouvez selectionner ces modes avant chaque partie, en appuyant sur
la touche N pour le mode normal et la touche C pour le mode crazy.

    Pendant la partie :
        - P     pause. Droite ou gauche pour redémarrer
        - ESC   Game Over
        - touche directionnelle droite/gauche pour déplacer le viseur
        - "Fire" pour lancer la pierre
Si vous utilisez un ControlPad CD 32, les deux boutons Forward/reverse peuvent
servir à affiner la visée.

    Les bonus :
        -Ils sont activés de deux manières. Lorsque vous les touchés avec
         une pierre ou lorsque ils sont reliés à une pierre de la même couleur
         que celle que vous venez de lancer, ils remplacent alors la couleur
         manquante et disparaissent avec les deux autres pierres.

        - Pour éviter les "mauvais" bonus essayez de détruire les pierres du
          haut pour les faire tomber sans les toucher.

        - Description des bonus/malus :
                         1 crédit
                         -1 crédit
                         200 points
                         niveau suivant
                         +1 ou -1 crédit

    Lorsque vous avez terminé une partie (Game Over) appuyez sur le bouton
"Fire" pour revenir à l'écran de départ puis appuyez sur ESC pour quitter
et sauver les HighScores.

Installation:

  Copiez simplement le répertoire BubbleStones ou vous voulez. Il n'y à pas
  d'assignation a faire.
  
  BubbleStones nécessite le système 2.0 ou plus.
  

Testé sur:
  - Amiga 1200, disque dur 170 Mo, system 3.0
  
  - Amiga 1200, MTEC 030/28Mhz, Fast 4Mo, Disque dur 210 Mo, System 3.0

  - Amiga 1200, Apollo 1240/40, Fast 32Mo, Disque dur 4.3Go, System 3.5

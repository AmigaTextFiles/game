INTRODUCTION

Thank you for downloading 2H4U v1.2.

This program has been tested and built on Windows. However, it should perfectly work on Linux, or any other OS supported by SDL.

Please let us know of any successful building with any other OS, as well as the changes you may have done to the program.

Please do not hesitate to tell us about any change brought to the game, in any field.

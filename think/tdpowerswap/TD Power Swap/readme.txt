 Simple logical game based on idea of Amiga game Pure Brain.
 Your task is to change order of pictures into oryginal one.
 You have limited time for each level. Number of levels: 32.

 During game there are visible level codes which can be typed
 on menu screen. The level code is changing start level.
 Code for fist level is 'reset'.

 Enjoy ;)

 !!! Game created on MorphOS !!!


## Requirements

 MorphOS 1.4
 PowerSDL v11.8


## Installation

 Copy game drawer anywhere You want.

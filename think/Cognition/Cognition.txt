COGNITION... THE 100 PERCENT VERSION

These 100 % versions of Cognition return the used memory after quitting the
game. Try the normal version first. The "LowMem" version was made for
Amigas with a small amount of (Chip) memory. By the way, did you already
know that the game is fully capable of multi-tasking? Try pressing the left
Amiga key together with the M key. Last but not least, AGA machines will
show the benefit of a smooth color range in the game's background.

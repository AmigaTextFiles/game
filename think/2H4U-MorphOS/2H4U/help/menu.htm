The Game Menu



Select an item by using the arrow keys (up or down) and press enter to start


PLAY

Start a new game !

MUSIC

Toggle the background music on/off.

SOUNDS

Toggle the sound effects on/off.

DISPLAY

Allows you to choose to play in fullscreen mode or windowed (800 x 600 ) mode.

SCORES

Displays the top 5 scores

QUIT

Quits the game

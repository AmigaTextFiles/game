[net-bubble 0.9.1]

==================== INSTALL ====================

UNIX: 
	tar -xzvf net-bubble-0.9.1.tar.gz
	cd net-bubble-0.9.1
	make
	./net-bubble

WINDOWS: 
	unzip net-bubble-0.9.1.zip
	net-bubble.exe


==================== OPTION ====================

FULLSCREEN:
	option "-fullscreen" in command line 

	
==================== CONTROLS ====================

NB: The controls are hardcoded in this version.

ONE PLAYER:
	LEFT - RIGHT - SPACE

TWO PLAYERS:
	LEFT PLAYER:
		S - D - SPACE

	RIGHT PLAYER:
		LEFT - RIGHT - ZERO (KEYPAD)

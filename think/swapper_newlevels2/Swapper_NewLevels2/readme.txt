
 Swapper - New Levels 2
 ----------------------


ENGLISH
-------

Swapper New Levels 2 is a freeware pack of 10 levels for the game Swapper.
It was created by David Brunet in 2006 using the builtin level editor
of Swapper.

Swapper is a game made by Jakub Brzezniak and available on :
http://uk.aminet.net/game/think/Swapper.lha

How to install the new levels ? Simply Copy all the new levels in the "Levels" drawer
of Swapper.

How to play it ? Run Swapper and select "Options" in the menu, then select
"Level..." and choose the level you want to play.

Enjoy :-)

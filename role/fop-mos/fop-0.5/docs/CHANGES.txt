CHANGES.txt for FOP: Fight or Perish
by Bill Kendrick <bill@newbreedsoftware.com>
http://www.newbreedsoftware.com/fop/

0.5 (2009.03.15)
- Fixed fine scrolling bug.
- Old hard-coded map size persisted; replaced all with dynamic
  map size.
- Players' initial positions can be set explicitly in the map.
- Exits added, which loads the next level after all players
  have exited.
- Game ends after the last level (cannot find map file).
- Level selection from Start Game screen (use PgUp/PgDn)
- Spaced HUDs so they don't overlap.

0.4 (2009.03.15)
- Added fine-scrolling.
- Added missing default controls for bombs.
- Created stats icons for use on player select screen and in HUD.
- Initial creation of different character graphics (ply1 only).
- Added score display to HUD.
- Each character has their own weapon graphics.
- Added temporary upgrade bonuses ("?" box).

0.3 (2009.03.01)
- Ability to override control settings via config file.
- Joystick input support.

0.2 (2009.03.01)
- True four-player support (with different key input,
  allowing simultaneous play).
- Player select screen.
- Some amount of documentation.

0.1 (2009.02.28)
- Four-player support.
- Different player speed, health, weapon attributes.
- Gave official name to the game.
- Arbitrarily-sized maps.

2009.02.26
- Added bitmapped graphics.
- Fixes & corrections.

2009.02.25
- Initial one-player gameplay, with colored rectangles as shapes.

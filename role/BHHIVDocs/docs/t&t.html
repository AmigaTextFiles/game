Tricks & Tips

How to do the little things. Not in any order

Selecting Weapons.
Should you find that you are unable to select any weapons for players or monsters, try pressing the 'Natural Attacks' button twice. This should fix the problem. This is also usefull if players are attacking with an empty hand, it is hoped that this bug will be sorted out in version 1.1

Time
Time can be advanced by pressing the space bar. Also you may switch advance modes with the 't' key, stepping through from Mins, Turns, Hour, Days, Real time and back to Mins again.

String Gadgets
Many of the string gadgets in the Player and Monster windows have some simple enhancements. The Hit Point and XP gadgets can handle addition and subtraction. The Stats on the monster window will auto-roll if set to '0'

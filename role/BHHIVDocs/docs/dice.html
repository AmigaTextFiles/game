Dice - IV

A Guide to this accessory for Bigby's Helping Hand - IV.

A small Die rolling utility for DM's in the know. Like potion and movement Dice will attempt to open on Bigby's Helping Hand -IV's screen first, then falling back to the Workbench screen.

There are a couple of self explanatory gadgets and a string gadget that accepts dice strings in the standard manner (ie: 2d4+7). As well as this dice will accept a dice string if run from the Shell.
